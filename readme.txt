环境搭建，先下载python开发环境，然后是pycharm编辑软件，
对Python中下载相应的包，选择适用的版本号，
对爬虫任务的绝对地址进行相应修改；
在pycharm最下面找到终端，输入python manager.py runserver 8080运行服务器
打开目前流行的大型浏览器，或者点击终端显示出来的地址进入本平台
